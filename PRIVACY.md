EduTrack (SIBA) Privacy Notice

This project is a demo and stores all data locally in the user's browser via localStorage. Please be aware:

- No server-side storage is provided; data exists only on the local device and may be lost if the browser cache/localStorage is cleared.
- Do not store passwords, personal identification numbers, or sensitive personal data in this application.
- If you intend to publish this project, implement secure authentication and server-side data protection.
